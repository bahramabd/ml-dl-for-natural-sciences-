# Machine Learning for Natural Sciences

Solved notebooks for the **Machine Learning for Natural Sciences** course at KIT

The notebooks cover core machine learning concepts and their application to natural-science problems, from mathematical foundations and classical ML models to neural networks, atomistic simulations, CNNs, and RNNs.

## Contents

| Exercise | Notebook | Main topics |
| --- | --- | --- |
| 1.5 | `ex_01_5_MathBasics.ipynb` | Distance functions, precision-recall curves, PCA, dimensionality reduction |
| 2 | `ex_02_Decisiontrees (1).ipynb` | Data preprocessing, decision trees, entropy, Gini index, random forests |
| 3 | `ex_03_MLBasics(1).ipynb` | Linear regression, train/validation/test splits, gradient descent, ridge regression |
| 4 | `ex_04_MLBasics2(1).ipynb` | Bayes' theorem, data analysis, Naive Bayes estimation |
| 5 | `ex_05_NN(1).ipynb` | Neural networks, forward passes, batches, weight initialization, organic solar-cell data |
| 6 | `ex_06_AtomMD(1).ipynb` | Molecular dynamics, learning energies and gradients, neural-network potentials |
| 7 | `ex_07_CNN(1).ipynb` | PCA, image data processing, CNNs, transfer learning, test-set classification |
| 8 | `ex_08_RNN.ipynb` | Time-series data, GRU, LSTM, ECG classification, sentiment analysis |
